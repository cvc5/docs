Binary Documentation 

The easiest way to use cvc5 is usually to invoke the cvc5 binary via the command line. The quickstart guide gives a short introduction on how to use cvc5 via the SMT-LIBv2 interface, but cvc5 also supports other input languages . The behavior of cvc5 can be changed with a number of options , and output tags allow to extract structured information about the solving process.

Alternatively, cvc5 features several APIs for different programming languages.
